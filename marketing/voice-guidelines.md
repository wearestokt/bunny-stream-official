This is the copy baseline for Stream Bunny marketing, Marketplace listings, and in-product text.

If you’re writing new copy, read this first. Then write like you mean it.

## Who Stream Bunny talks to

- Creative companies.
- Studios. Filmmakers. Agencies.
- Design teams shipping portfolio sites, reels, and case study pages.
- Indie designers building pages that feel like product.

Not:
- Marketing gurus.
- Funnel builders.
- “Video CMS solution” shoppers.

## The voice rules

Keep these tight. No exceptions.

1. **Short sentences.**
2. **Confident, slightly cocky.** Not arrogant. Not cringe.
3. **Specific beats vague.** Say “Player + Controls” instead of “flexible system.”
4. **We say “drop.”** Not “integrate.” Not “seamlessly connect.”
5. **We say “iframe ceiling.”** Not “limitations of the embed.”
6. **No hype words.** If it reads like LinkedIn, delete it.
7. **No exclamation marks** unless it actually earns it. Most don’t.
8. **Price is a line item.** Not a pitch.
9. **Open source stays visible.** GitHub link belongs in the copy.
10. **Built for creative companies** should be felt in every paragraph.

## Word lists

### Use

drop · ship · paste · render · restyle · swap · build · skin · raw · real · layout · canvas · player · controls · state · HLS · adaptive · renditions · templates

### Don’t use

leverage · supercharge · unlock 10x · revolutionary · game-changing · synergy · journey · ecosystem · solution · next level · effortlessly · transform your workflow

## Headline patterns

Pick a pattern. Don’t freestyle a manifesto.

1. **Verb the noun.**
   - “Kill the iframe ceiling.”
   - “Build real video UI.”

2. **What you’d build if X.**
   - “What you’d build if Bunny Stream wasn’t an iframe.”
   - “What you’d build if controls lived in your layout.”

3. **Noun. Built like noun.**
   - “Bunny Stream. Built like UI.”
   - “Video controls. Built like components.”

4. **Two hard facts.**
   - “HLS · adaptive quality.”
   - “Player + Controls · shared state.”

## Punctuation + formatting tics

- **Lowercase headers are fine.** Keep them blunt.
- **Em dashes** are fine—use them for punch, not rambling.
- **Mid-dot separator** is the brand tic: `·`
  - Example: `$49 once · lifetime updates`
- **No Oxford comma required.** Don’t overthink it.
- **No exclamation marks** by default.
- Use **bold** for anchors (pricing, Pro name, Player + Controls).

## Pricing wording

Always write it exactly like this:

**$49 once · lifetime updates**

Never:
- “just $49”
- “only $49”
- “$49 lifetime deal”
- “limited-time”

## Free tier wording

Always:

**5 free canvas inserts per workspace**

Never:
- “limited trial”
- “evaluation period” (unless you’re quoting an internal dev note—don’t)
- “free for a limited time”

## The non-negotiables

Every public page, listing, or email should keep these true:

1. **Built for creative companies.**
2. **Open source on GitHub** (`https://github.com/wearestokt/bunny-stream-official`).
3. **Pro is one-time**: **$49 once · lifetime updates**. No subscription framing.
